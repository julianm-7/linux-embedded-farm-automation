# CIA Sponsored Linux Embedded-system Aquaponic Farm Automation for crop identification


This project is a Linux-based embedded farm automation system integrating **C++ motor control**, **Velodyne VLP-32C LiDAR**, and **cloud-connected machine learning** for crop analysis. It is designed to fully cover a **182.88 cm Ã— 121.92 cm aquaponic bed** in a serpentine grid pattern.
